Codeception 快速开始

Codeception是一个马上就能使用的PHP测试框架。它能快速的安装并且不需要其它任何依赖(只需要PHP运行环境)。只需要简单的一个配置步骤你就能马上开始测试你的Web应用程序。  

1. 下载

下载然后保存 phar文件到Web应用程序的根目录。 

下载 Codeception

或者通过 console 下载

wget http://codeception.com/codecept.phar

如果你使用的是PHP 5.4或者5.5 < 5.5.9

下载 Codeception

或者通过 console 下载

wget http://codeception.com/php54/codecept.phar
通过 Composer进行安装 
php composer.phar require "codeception/codeception:*"

在这种情况下使用./vendor/bin/codecept 来替代 codecept.phar .

全局安装 
sudo curl -LsS http://codeception.com/codecept.phar -o /usr/local/bin/codecept
sudo chmod a+x /usr/local/bin/codecept

使用 codecept 替代 codecept.phar .

2. 安装

打开命令行工具Cd 到 codecept.phar文件所在目录，然后执行下面的语句:

php codecept.phar bootstrap

系统将会 codeception.yml文件 和   tests目录.

3.创建测试脚本

创建你的第一个验收测试脚本.验收测试脚本模拟一个真实的用户去访问你的网站.

php codecept.phar generate:cept acceptance Welcome

Codeception 所有场景测试都被统称为:Cepts.

4. 写一个基础的测试文件

现在我们编写你的第一个测试脚本.编辑我们刚刚创建的文件 tests/acceptance/WelcomeCept.php

{% highlight php %} wantTo('ensure that frontpage works'); $I->amOnPage('/'); $I->see('Home'); ?>{% endhighlight %}

它将会检测你的前端页面是否包含 首页这2个文字.

5. 配置验收测试

请确保您本地的 开发服务器是正常运行的.  把Web应用程序的URL地址加入到下面的文件中:  tests/acceptance.suite.yml 

class_name: AcceptanceTester
modules:
    enabled:
        - PhpBrowser:
            url: {您的网站程序URL地址}
        - \Helper\Acceptance

6. 运行!

通过下面的命令运行Codeception

php codecept.phar run

它们将会通过 PhpBrowser执行我们的Welcome测试脚本.这个PHP测试脚本会检测HTML的内容，点击链接，填充表单然后提交POST或者GET请求.更多更加复杂的测试脚本会依赖于WebDriver模块.

如果你把上面的操作都成功做了一遍，并且您的应用程序前端页面有"首页"这2个字,您将会看到下面输出的内容:
Suite acceptance started 
Trying to ensure that frontpage works (WelcomeCept.php) - Ok
Suite functional started
Suite unit started

Time: 1 second, Memory: 21.00Mb
OK (1 test, 1 assertions)

通过学习 验收测试章节 您将会学习到如何模拟一个真实的用户去对您的网站进行点击链接, 填充表单这样的操作.
